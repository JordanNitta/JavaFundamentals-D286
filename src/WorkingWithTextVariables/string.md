#### String - meant for dealing with text
#### Some string methods
```java
// Return the length
length();

// This method returns the character at the specified index in the String.
charAt();

//The indexOf method returns the index of the first occurrence of the specified String within the calling String.
//If not found it returns -1.
indexOf(str);

//This method returns a new String that is a substring of the calling String, starting at the specified beginIndex and extending to the end of the String or the specified end index.
// Exclusive 
        
substring(3, 5);
substring(3);

// These methods return a new string that is a copy of the calling string, with all the characters converted to lowercase / uppercase respectively.
toLowerCase();
toUpperCase();

// The trim method returns a new String that is a copy of the calling string, with leading and trailing white space removed.
trim();

```
